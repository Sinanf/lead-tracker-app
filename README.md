# Lead Tracker App

A lightweight mobile-first web app for saving and managing sales leads. Leads are stored in localStorage so they persist between sessions — no backend required.

## Live Demo

[lead-tracker-app on Vercel](https://lead-tracker-app.vercel.app)

## Features

- Add and save leads instantly
- Persistent storage via localStorage
- Delete all leads with one tap
- PWA-ready (installable on mobile via `site.webmanifest`)
- Minimal, fast, zero dependencies

## Tech Stack

Vanilla JavaScript · HTML · CSS  
No frameworks, no build step — open `index.html` and it works.

## Getting Started

```bash
# No install needed — just open in browser
open index.html
```
